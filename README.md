# Postali

Búsqueda de códigos postales de México por colonia.

## Desarrollo

```sh
npm install
cp .env.example .env
# Editar .env con PUBLIC_SUPABASE_URL y PUBLIC_SUPABASE_ANON_KEY
npm run dev
```

## Build

```sh
npm run build
```

Salida estática en `dist/` (apta para GitHub Pages).

## Variables de entorno

- `PUBLIC_SUPABASE_URL` — URL del proyecto Supabase
- `PUBLIC_SUPABASE_ANON_KEY` — Clave anónima (anon key) de Supabase

Ver `.env.example`.
